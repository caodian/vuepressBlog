## CSS

### 盒模型
 盒模型分为标准盒模型和IE盒模型两种，相关属性有 width（内容）、padding、border、margin
1. 标准盒模型
   标准盒模型的width就等于内容宽度
2. IE盒模型
   IE盒模型的width = 内容宽度 + padding + border
   如下图对比所示：
   ![盒模型对比图](./images/learn_1.png)

|盒模型|width|height|
|:---:|:---:|:---:|
|IE盒模型|内容宽度+padding+border|内容高度+padding+border|
|标准盒模型|内容宽度|内容高度|
